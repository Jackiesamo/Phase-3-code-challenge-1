About This Project

This is where you can find your Phase 3 Python. It has practical labs and activities that aim at developing effective skills in:

  Conditional Statements
  Loops
  Object-Oriented Programming
  Properties & Decorators
  Counting Sentences
  Cash Register Implementation.

[?] Requirements

To operate this project without problems, you need to have:

 Python 3.12+
 venv or pipenv to operate a virtual environment.
 pytest for running tests

Getting Started

To install the project in your machine, follow the following steps:

1[?] Clone the repository

git clone <your-repo-url>

2[?] Browse into the project folder.

cd <your-project-folder>

3[?] Develop and start up your virtual world.

pipenv install
pipenv shell

4[?] Run the test suite

pytest
